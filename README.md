📝 Task Manager App
A simple and elegant Task Manager built with React to help you organize your daily tasks efficiently.

🌟 Features

* ✅ Add tasks
* 🗑️ Delete tasks
* 📋 View tasks instantly
* ⚡ Fast and responsive
* 🎯 Clean UI

🛠️ Tech Stack

* ⚛️ React.js
* 💻 JavaScript
* 🎨 CSS

⚙️ Installation

```bash
git clone https://github.com/Kavya2502/Task-Manager-App.git
cd Task-Manager-App
npm install
npm start
```
